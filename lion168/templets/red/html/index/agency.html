金狮国际合作计划协议书


1. 我们的权利和义务

金狮国际是全球最著名、运营最成功 的线上数字娱乐巨头之一，在为广大用户提供优质的产品和专业客户服务过程中，建立了极佳的声誉。 公司的财政来源主要是获得来自全球娱乐和游戏企业家的支持。

1.1 登记代理账户
我们会登记您的代理账户并追踪您的下线玩家在网站上进行游戏的过程。为了遵循我们定期设定或编制的一些基本要求，在必要的时候我们保留拒绝或者关闭其账户的权利。


1.2 追踪代理下线玩家的投注
我们将追踪您的下线玩家的投注活动，我们将定期检查并保留拒绝任何玩家违反我们的游戏规则进行申诉的权利。详情请查阅金狮国际的条款与条件。

1.3 支付代理佣金
详情见金狮国际“佣金协议”相关条款。

1.4 修正
我们可以随时修改或替换本协议内的任何规则与条例，我们会将变更及时公布在官方上。修正范围包括：变更代理佣金和合作计划的规则，在我们网站发布了变更通知或者修正了新协议后，您继续参与合作计划，将被视为接受更改或者新协议的约束力。

2. 您的权利和义务

2.1 代理账户开设
如果您同意参与此合作计划,我们通过审核后即可开通您的代理账户，并给您开设相应的代理链接，您发展的玩家只需在此代理链接登录注册账户即可或者您可以直接为您的下线玩家开通游戏账户进行游戏。

2.2 代理指定
通过此协议，我们授予您非独特的、不可归属的权利以便依据本协议的规则和条款来推荐客户到任何我们的网站，同时我们愿意随时与您合作并从您处获得其他协助以便改善我们相同或类似属性的服务。出于商业机密，您不应授权其他人来申领您的代理佣金或其它奖偿。


2.3 诚信
出于诚信，无论实际上是否会使我们造成损失，您都不可以蓄意从已知的或涉嫌的交易中获利。您必须遵守此协议，否则我们可以凭借此合理的理由相信此交易无效，我们将保留或扣留所有款项的权利。


2.4 代理的责任
金狮国际合营计划旨在您的直接参与。您不应以其他参与者的名义开设帐户。我们不接受任何为第三方开设代理帐户、或转让代理帐户的行为。您如果要转让一个帐户给另一个有利益的帐户拥有者，必须联系我们以获得许可才能进行。此许可仅基于我们的单方面考虑之上。

 
2.5 机密信息
在此协议期间，您可能被信赖于有关我们商业的、运作的、或潜在的技术以及合作项目的机密信息（例如包括您在 合营计划下所获得的客户酬金）。您必须同意避免暴露这些信息和避免未经许可授予第三方和其它方使用这些机密信息的行为，除非您在之前获得了我们的书面承诺，或者您使用这些机密信息仅仅只是为了进一步达成此协议必要目的。在此协议终止后，您关于尊重这些机密信息的责任和义务依然生效。


3. 代理佣金计算

3.1 代理佣金计算
您将获得我们在体育博彩和真人娱乐场里从您所介绍的玩家中所赚取的纯利润的25% - 50%的收益。
纯利润的定义如下：纯利润 =体育博彩+娱乐场盈利-玩家红利-退款-玩家存提款費用-平台费。纯利润×佣金比例=您当月合作伙伴佣金。
依照条款1.4，我们将保留变更合营酬金的计算比率以及计算方法的权利。
如果发现任何的欺诈行为，我们有权停止支付您代理佣金。

3.2 代理佣金计算表
每一个金狮国际合营账户必须有最少5位活跃玩家，并且每月最低佣金费用支付要求是人民币500元， 当月不足500元的代理， 自动累积进入下一个月。

3.3 升级奖金和业绩审核
金狮国际市场团队会根据代理客户每月的业绩进行升降级的调整，同时根据代理当月产生的纯利润的金额代理佣金比例也同步有提高， 最高可达50%。

3.4 代理佣金的计算
每月佣金最快将会于每月的第一个周日进行结算，经核数完毕后佣金将于次日存入合作伙伴所提供的指定账户内，每月佣金最低必须达到500元人民币，低于500元的代理佣金将会累积到下一个月的佣金一起发放。
如果有玩家违反我们的条约或有任何的欺诈行为，我们有权关闭该玩家账户，并将从代理账户中扣除该玩家中所获得的净收益。
我们要求合营客户的总月酬金的30%不能只来自于一个活跃玩家的净盈利。

4. 协议期限和终止
4.1 期限和终止
当您获准成为金狮国际合作伙伴时，此协议将开始生效，直到任何一方通告另一方希望终结此协议，在这种情况下此协议将被立即终止。合同终结仅取决于任何一方的意愿而无论是否有原因。

4.2 多重帐户和自我推荐

合作伙伴本身的游戏账号不可以在自己的合作伙伴旗下。如有上述情况发生，我们将会扣除该账户所得到的输赢，并将该账户从您合作伙伴下线里注销。如果代理下线多个活跃玩家出现同一个IP，则认定为一个活跃玩家。


4.3代理合作关系
我们和您是独立的合作方， 此合约的任何部分都将不会衍生出伙伴关系、联合企业、中介、特许权、销售代表、或存在在我们之间的雇佣关系。您将没有权利以我们的名义制作和接受任何提议或请求。


5. 免责声明
有关于代理合作项目、我们自己和代理佣金的付款安排，我们没有任何明指的或暗指的陈述或保证，也不明指和暗指任何一个产生于履行、涉及和贸易用途过程的担保。另外，我们不发表任何关于我们网站运作将会无干扰和无错误的声明，因而即便一旦出现问题我们也将对此不负任何责任。如果在金狮国际代理页面所提供的系统报告和金狮国际数据库所提供的报告之间有差异的话，以我们数据库的数据为准。

6. 自主的调查
您认可您已经阅读了此协议并同意其所有的规则和条款。您理解我们可能会在任何时候（直接或间接地）修改或者变更我们的合作协议，您已经自主评估了参与的意愿并不依赖于任何在此协议之外的言论、保证和声明。

7. 其它事项

7.1 法律条例
菲律宾的法律将管辖此协议，任何与此协议相关的法律行为必须被带到菲律宾，同时您必须不可改变地认可菲律宾法庭的审判权。
金狮国际体育博彩以及娱乐场产品将在菲律宾第一卡格杨（First Cagayan）地区国际娱乐执照的严厉监管下操作。

7.2 归属性
您不可以在我们的书面许可之前通过法律条例运作签署此协议。根据这一约束，此协议将受制于、利益生效于、并可强行实施于您和我们以及我们各自的指定人之间。

7.3 坚持权
如果我们无法强制您严格执行此协议，这并不构成我们对随后强制实施此协议的规定或任何其他规定权利的放弃。此协议不允许有修改、附加、删除和插行补充，否则我们将不予认可。

7.4 补救
我们在此记载的权利和补救不是相互独立的，也就是说一个或多个此协议规定的操作不妨碍任何其它规定的运作。您认可、确认并同意：由违背及先兆违背此协议所造成的危害是不适当的。在违背和先兆违背此协议任何规定的情况下，我们可能通过特定的表现、指令或其它公正的补救方式来寻求强制执行和遵守。此协议中没有任何东西可以限制或影响我们在法律上的权利，对于违背和先兆违背此协议规定，我们的权利应是公正的并在法律上是可强制执行的，而这个规定的目的是为了让我们的权利清晰化。

7.5 可分离性/放弃

在任何可能的情况下，此协议的每条规定应在可应用的法律下被解释为有效和合理的，但是，如果任何此协议的规定在任何方面被法律认定是不合理的、非法的或是不可执行的，那么此规定将仅对于此无效性和不可执行性的程度来说是无效的，而并不是说此协议中的其它和任何关于其它的规定是无效的。如果在指导或在强制实施任何权利失败时不意味着弃权，不弃权必须以书面的形式才能生效。
